## üõ† Common Commands

### Building
```bash
../ktroid.py build debug          # Debug APK
../ktroid.py build release        # Release APK (signed)
../ktroid.py build bundle         # Android App Bundle
../ktroid.py clean                # Clean build
```

### Device Management
```bash
../ktroid.py emulator list        # List AVDs
../ktroid.py emulator start       # Start emulator
../ktroid.py uninstall            # Remove app from device
```

### Testing
```bash
../ktroid.py test                 # Run unit tests
../ktroid.py test instrumented    # Run instrumented tests
```

### Version Management
```bash
../ktroid.py bump code            # Increment version code
../ktroid.py bump name            # Increment version name
../ktroid.py bump both            # Increment both
```

### Dependencies
```bash
../ktroid.py dep-list             # List all dependencies
../ktroid.py dep-remove <name>    # Remove dependency
```

### Release Preparation
```bash
../ktroid.py signing              # Setup signing config
../ktroid.py bump both            # Update version
../ktroid.py build release        # Build signed APK
```
